# AI 每日速递 | 2026-06-12

## 🚀 今日热点

### OpenAI 更新 Responses API，原生支持 Agent 多步骤任务

OpenAI 于 6 月 11 日更新 Responses API，新增对 Agent 任务的原生支持，允许模型自主规划、调用工具并完成多步骤任务，无需开发者手动编排复杂工作流。

同时，OpenAI 将高风险 `GPT-5.3-Codex` 请求自动路由至更稳定的 `GPT-5.2` 模型执行，以平衡代码生成能力与安全性。这一"模型路由"策略体现了 OpenAI 在生产级 AI 编程工具上的稳健化思路。

**核心亮点：**
- Responses API 现在原生支持 Agent 任务规划与工具调用
- 高风险请求自动降级至更稳定的 GPT-5.2
- 标志着 OpenAI 从「模型能力」向「平台可靠性」的战略重心转移

### Anthropic 发布 Windows 版 Claude Cowork，YOLO 模式上线

Anthropic 正式将 Claude Cowork 扩展至 Windows 平台，结束了自 1 月发布以来仅限 macOS 的局面。Windows 版功能与 macOS 完全对齐，支持对话式编程、代码库理解与自动修改。

与此同时，Claude Code Desktop 新增 **YOLO 模式**（You Only Live Once），允许 Agent 在无需逐条确认的情况下自动执行代码修改、文件操作等任务，大幅提升开发效率。该模式面向信任 Agent 输出的高级用户，减少了人机交互摩擦。

**核心亮点：**
- Windows 版 Claude Cowork 正式发布，功能完全对齐 macOS
- YOLO 模式：跳过逐条确认，Agent 全自动执行
- Anthropic 加速桌面端 AI 编程工具的平台覆盖

### 华为云提出 Agentic Infra 新范式，发布 10 万卡级智算集群

华为云在 INSPIRE 创想者大会上提出 **Agentic Infra** 新范式，重新定义 AI 时代的基础设施架构。核心发布包括：

- **AICS 灵衢智算集群**：支持 10 万卡规模、200 EFLOPS 算力
- **AMS Agentic 记忆存储**：专为 Agent 长期记忆设计的高性能存储
- **CCE Volcano Next 引擎**：下一代容器调度引擎，优化 Agent 工作负载
- **AgentSphere 弹性沙箱**：安全隔离的 Agent 运行环境
- **ModelArts Next 平台**与企业级智能体平台**智果 AgentArts**

开源版本 `openJiuwen` 与华为云版本同源度超 90%，彰显了其开放生态战略。

**核心亮点：**
- 业界首个面向 Agent 时代的基础设施参考架构
- 10 万卡级集群 + 专用 Agent 记忆存储
- 开源版本同源度 90%+，推动产业标准共建

### 阶跃星辰 Step 3.7 Flash 登顶 Artificial Analysis 三项第一

阶跃星辰发布 `Step 3.7 Flash` 模型，在 Artificial Analysis 榜单的**速度、性价比、端到端**三项指标均排名第一。

关键数据：
- 输出速度最高 **416 tokens/s**

这标志着国产大模型在「性价比」赛道上已建立起全球竞争力，对 API 商用市场格局将产生实质性冲击。

### 微信开放 AI 智能体平台，美团滴滴携程首批内测

微信宣布开放平台能力，允许开发者主动授权接入微信 AI 生态，使智能体能够调取小程序完成交易和服务。美团、携程、同程、滴滴作为**首批内测应用**，已与微信团队联合开发并测试接入。

这是中国互联网史上首次「超级 App」向第三方 AI Agent 开放核心交易能力，标志着微信从「连接器」向「Agent 平台」的战略转型，也意味着 AI 入口之争从搜索场景延伸至高频交易场景。

**核心亮点：**
- 微信首次向第三方 AI Agent 开放小程序调用能力
- 美团/携程/滴滴首批接入，覆盖本地生活核心场景
- 超级 App 的 Agent 平台化趋势正式开启

### 阿里通义千问发布高考志愿填报 Agent

阿里通义千问发布**高考志愿填报 Agent**，具备主动对话能力，可模拟人类志愿规划师的参谋角色。千问产品负责人郑嗣寿表示，Agent 定位为辅助决策，而非取代专家判断。

此举与微信智能体生态开放形成呼应——AI 入口之争正从通用搜索向**垂直场景**快速延伸，教育、医疗、法律等高价值场景成为各家大模型厂商的必争之地。
